# CalcURisk (Stable Release v1.0)

This is the stable backup version of CalcURisk.

## Features
- Mobile-friendly layout (iPhone tested)
- Scroll-enabled to avoid clipping
- Pattern dropdown sorted alphabetically (includes RCT)
- Pull-down reset (does not clear saved trade plans)

## Usage
- Open `index.html` in browser
- Or upload to GitHub Pages

## Notes
This version is considered stable for testing and deployment.
